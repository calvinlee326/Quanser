# Quanser QCar — Self-Driving Car Lab

Experiments and lab work for autonomous driving using the Quanser QCar platform. Covers lane detection, computer vision, and steering control algorithms.

## Contents

| Folder | Description |
|--------|-------------|
| `Lab/` | Camera tests, lane detection scripts, and driving experiments |
| `SDCS/` | Self-driving car system code |
| `examples/` | Reference examples |
| `libraries/` | Shared utility libraries |
| `concept_reviews/` | Notes and concept breakdowns |
| `user_manuals/` | Hardware and software manuals |

## Tech Stack

- Python 3
- OpenCV
- Quanser QUARC / HAL libraries
- Intel RealSense (depth camera)

## Key Scripts

- `Lab/lane_detect.py` — Lane detection using computer vision
- `Lab/drive_in_circle.py` — Circular path driving test
- `Lab/intel_realsense_lane.py` — Lane detection with depth camera
